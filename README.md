那这个仓库是为你写的。

## 数学概念索引

> 这里专门解释公式。每一个条目都包含：
> - 这个符号/公式在说什么（大白话）
> - 一个具体的数字例子，手算一遍
> - 它在机器学习里用在哪里
>
> **目标：初中数学基础的人看完之后能说出"哦，原来是这个意思"。**

### 基础符号

- [ ] `∑`（求和符号）— 就是把一堆数加在一起
- [ ] `||x||`（范数）— 向量的"长度"是多少
- [ ] `log`（对数）— 指数的反过来
- [ ] `e`（自然常数）— 为什么机器学习里到处都是它

### 微积分概念

- [x] [微分与偏导数](math/calculus-diff-and-partial-derivatives.md) — 从斜率逼近到偏导数，以及它们如何驱动梯度下降
- [ ] 梯度 — 多维空间里的"最陡方向"
- [ ] 链式法则 — 函数套函数时怎么求导（反向传播的数学基础）

### 概率与统计

- [ ] 概率 `P(A)` — 基础复习
- [ ] 条件概率 `P(A|B)` — 已知B发生，A发生的概率
- [ ] 期望值 — 平均结果是多少
- [ ] 正态分布（高斯分布）— 为什么到处都是钟形曲线

### 线性代数

- [ ] 向量 — 一组数，可以理解成"一行特征"
- [ ] 矩阵乘法 — 为什么神经网络用矩阵来算
- [ ] 转置 — 行变列，列变行，有什么用

### 机器学习常用公式

- [ ] 均方误差（MSE）— `(预测值 - 真实值)²` 的平均，最常见的损失函数
- [ ] 交叉熵损失 — 分类任务用的损失函数，为什么比MSE好
- [ ] Sigmoid 函数 — `1 / (1 + e^(-x))`，把任意数压到 0~1 之间
- [ ] Softmax 函数 — 把一组数变成概率分布
- [ ] 参数更新公式 — `W = W - 学习率 × 梯度`，梯度下降的核心一步
