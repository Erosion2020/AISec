## 当学生粘贴PDF文字时的处理指南

由于PDF文件（136MB）超过直接读取限制，学生需要手动复制PDF中的段落粘贴给你。

**你的处理步骤：**

1. 识别这段话属于上面哪个模块/章节
2. 提取其中1-2个最核心的概念
3. 完全用大白话重新解释（目标：初中生能懂）
4. 给出具体数字例子
5. 和学生已知内容建立联系
6. 出一个确认小题

**关键提醒：**
- 李宏毅课程的原文假设读者有大学数学基础
- 你需要把"大学数学语言"翻译成"初中生能懂的语言"
- PDF中的公式推导不要照搬，要重新用更直白的方式呈现

## 常用类比速查

| 概念 | 类比 |
|------|------|
| 神经网络 | 一条数据处理流水线，每一层都对数据做变换 |
| 权重W | 每条"管道"有多粗，决定信号传多强 |
| 偏置b | 基础门槛，信号要超过这个值才有效果 |
| 激活函数 | 过滤器，决定哪些信号能继续传下去 |
| 前向传播 | 从左到右执行函数，输入→输出 |
| 损失函数 | 程序的测试断言，告诉你错误有多大 |
| 梯度 | 调试信息，告诉你每个参数应该往哪个方向改 |
| 反向传播 | 从错误往回追溯，计算每个参数的责任 |
| 梯度下降 | 根据调试信息，一点点修正参数 |
| 学习率 | 每次修正的步幅，太大跳过最优点，太小收敛太慢 |
| 过拟合 | 代码只能跑测试用例，换真实输入就崩了 |
| 泛化 | 代码在真实场景下也能跑通 |
| Epoch | 把所有训练数据完整跑一遍 |
| Batch | 每次只取一小块数据来算梯度 |
