# 微积分：变化率的工具

微积分研究一件事：**某个量在变化，变化有多快？**
所有的微积分工具，都是围绕这个问题展开的。
在 ML 里，"变化有多快"就是"调整参数能让损失减少多少"——这就是梯度下降的根基。

---

## 导数：变化有多快

**直觉**：导数告诉你，当你把输入改变一点点时，输出会改变多少。
更具体地说：输入改变了 1，输出改变了几？这就是导数。

**几何含义**：导数是函数曲线在某一点的切线斜率。斜率越大，变化越快。

**例子**（手算）：

函数 f(x) = x²

- 当 x = 2 时，f(2) = 4
- 当 x = 2.001 时，f(2.001) ≈ 4.004001
- 输入变化了 0.001，输出变化了约 0.004
- 变化比例 ≈ 0.004 / 0.001 = 4

所以在 x = 2 这个点，导数约等于 4。
（精确值：f(x) = x² 的导数是 2x，在 x=2 时是 2×2 = 4）

**ML 里的用法**：损失函数对某个参数的导数 = 该参数改变一点点时，损失会变化多少

---

## 偏导数：只改变一个变量

**直觉**：当函数有多个输入时，固定其他所有输入不动，只看其中一个输入变化时，输出怎么变。

**例子**（两个参数的损失函数）：

L(w, b) = (w×2 + b - 5)²，当前 w = 1，b = 0

- 对 w 的偏导数：固定 b = 0，只改变 w。
  L = (2w - 5)²，对 w 求导 = 2×(2w-5)×2 = 4(2w-5)
  代入 w=1：4×(2-5) = 4×(-3) = -12
  意思：把 w 增大一点，损失会减少（斜率为负，反向是增加的方向）

- 对 b 的偏导数：固定 w = 1，只改变 b。
  L = (2 + b - 5)² = (b-3)²，对 b 求导 = 2(b-3)
  代入 b=0：2×(-3) = -6

**ML 里的用法**：反向传播就是对每一个参数分别求偏导数

---

## 梯度：所有偏导数打包在一起

**直觉**：梯度是一个向量，它包含了函数对每个参数的偏导数。
梯度的方向 = 函数增长最快的方向。
**梯度的反方向** = 函数减小最快的方向。

**例子**（接上面）：

L(w, b) 的梯度 = [对w的偏导, 对b的偏导] = [-12, -6]

梯度告诉我们：
- w 方向上，损失变化的斜率是 -12
- b 方向上，损失变化的斜率是 -6

沿梯度反方向走一小步（w 增大，b 增大），损失会下降。

**ML 里的用法**：梯度下降就是沿梯度反方向更新所有参数：
`参数 = 参数 - 学习率 × 对该参数的偏导数`

---

## 链式法则：嵌套函数的导数

**直觉**：如果 y 依赖于 u，u 又依赖于 x，那么 y 关于 x 的变化率 =「y 对 u 的变化率」×「u 对 x 的变化率」

用一句话说：**外层变化 × 内层变化**

**例子**：

y = (3x + 1)²，想求 y 对 x 的导数

- 把它拆成两层：令 u = 3x + 1，则 y = u²
- y 对 u 的导数 = 2u
- u 对 x 的导数 = 3
- 链式法则：y 对 x 的导数 = 2u × 3 = 6u = 6(3x + 1)
- 代入 x=1：6×(3+1) = 24

**ML 里的用法**：反向传播的数学本质就是链式法则。
神经网络是多层函数嵌套，误差从输出层往回传播时，每层都在做"外层变化 × 内层变化"。

---

## 遇到微积分问题时的处理原则

1. 先问"这个导数在说什么量变化有多快"
2. 用 1~2 个参数的最简单例子手算
3. 建立它和 ML 场景（梯度下降 / 反向传播）的联系
4. 最后才涉及更一般的数学表达
