# 概率论：对不确定性的量化

概率论研究一件事：**当你不确定某件事会不会发生时，怎么用数字来描述这种不确定性。**
在 ML 里，模型的输出往往不是"这一定是猫"，而是"这是猫的概率是 87%"——这就是概率思维。

---

## 概率：对可能性的信念程度

**直觉**：概率是一个 0 到 1 之间的数，表示你对某件事发生的"信心"。
- 0 = 完全不可能发生
- 1 = 一定会发生
- 0.7 = 比较可能，但不确定

概率不是物理上"真实的频率"，而是**在现有信息下的合理判断**。

**例子**：
- 明天下雨的概率 = 0.6，意思是根据天气数据，下雨比不下雨可能性更大
- 模型判断一张图是猫的概率 = 0.92，意思是模型非常确信，但还留有 8% 的不确定性

**ML 里的用法**：分类问题的输出就是概率（用 Softmax 把数字变成 0~1 之间，且所有类别相加等于 1）

---

## 概率分布：描述所有可能结果

**直觉**：当一件事有多种可能的结果时，概率分布告诉你每种结果各自发生的概率。
所有结果的概率加起来必须等于 1。

**例子**（分类问题）：

模型看了一张图，输出三个类别的概率：
```
猫：0.75
狗：0.20
鸟：0.05
总和：1.00
```
这就是一个概率分布。0.75 + 0.20 + 0.05 = 1，符合规则。

**ML 里的用法**：
- Softmax 输出的就是一个概率分布
- 生成模型（如 Diffusion）学的是数据的概率分布

---

## 条件概率：已知A时，B发生的可能性

**直觉**："在已经知道某件事的情况下，另一件事发生的概率是多少？"

写法：P(B | A)，读作"在 A 发生的条件下，B 发生的概率"

**例子**：

- P(明天下雨) = 0.3（无任何信息时）
- P(明天下雨 | 今天有乌云) = 0.8（已知有乌云时）

已知有乌云，下雨的可能性大幅上升。条件改变了概率。

**ML 里的用法**：语言模型预测下一个词的过程就是条件概率
——"在前面已经出现'今天天气'的情况下，下一个词是'好'的概率是多少？"

---

## 期望：对所有可能结果的加权平均

**直觉**：期望是"平均来说，你预计会得到的结果"。
计算方式：每种结果的值 × 它发生的概率，全部加起来。

**例子**：

掷骰子，每个点数出现概率相等（各 1/6）：
```
期望 = 1×(1/6) + 2×(1/6) + 3×(1/6) + 4×(1/6) + 5×(1/6) + 6×(1/6)
     = (1+2+3+4+5+6) / 6
     = 21 / 6
     = 3.5
```

掷一次骰子，平均来说你会得到 3.5 点。

**ML 里的用法**：损失函数通常是对所有训练样本的期望损失——"平均来说，模型在这批数据上的错误有多大"

---

## 最大似然：让数据"最有可能出现"的参数

**直觉**：假设你已经观测到了一些数据，现在要找模型参数。
最大似然的思路是：**找那组参数，让你观测到这些数据的概率最大。**

换句话说：什么样的世界最可能产生你看到的这些数据？

**例子**：

你抛了 10 次硬币，得到 7 次正面、3 次反面。
问：这枚硬币正面朝上的概率 p 是多少？

最大似然的答案：选 p = 0.7，因为在 p = 0.7 时，"7次正面3次反面"出现的概率最大。

**ML 里的用法**：
- 训练神经网络本质上就是最大似然估计
- 交叉熵损失函数 = 负对数似然（最小化交叉熵 ≡ 最大化似然）

---

## 遇到概率问题时的处理原则

1. 先确认"不确定的是什么"（随机变量是什么）
2. 用简单的离散例子（如硬币、骰子）建立直觉
3. 建立它和 ML 场景（Softmax、损失函数、生成模型）的联系
4. 再讨论连续概率分布（如高斯分布）
